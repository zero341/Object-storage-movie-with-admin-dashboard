# Movie Object Storage System

This project provides a backend and frontend for a complete object storage movie management system with an admin dashboard. It allows for movie uploads, tracking, and management through a user-friendly interface.

## Features
- Upload movies
- Manage movies
- User authentication
- Input validation

## Installation
1. Clone the repository
2. Install dependencies 
   ```bash
   npm install
   ```
3. Configure your environment variables in the `.env` file.
4. Start the server 
   ```bash
   node server.js
   ```

## API
- Movies: `/api/movies`
- Uploads: `/api/uploads`
